# 📝 Interactive Quiz Application

### 🚀 Overview
This is a lightweight and interactive Quiz Application developed as part of my academic learning. The app is designed to test users' knowledge through a series of questions with real-time feedback.

### ✨ Features
- **Dynamic Questions:** Engaging multiple-choice questions.
- **Score Tracking:** Instant result calculation at the end of the quiz.
- **User-Friendly UI:** Simple and clean interface for a smooth user experience.
- **Privacy-Focused:** No personal data collection required.

### 🛠 Tech Stack
- **Framework:** Flutter/ Dart
- **IDE:** VS Code / Android Studio
- **Platform:** Android / Web


### 🏁 How to Run
1. Clone this repository.
2. Run `flutter pub get` to install dependencies.
3. Use `flutter run` to launch the app on your emulator or device.
